Tessel Repo Manager

This tool provides git-backed session persistence for compiled Tessel documents. Sessions are stored under sessions/<doc-slug>/<timestamp>/ in your repository.

Session package format:
session.json — field values + parameter values + notes tree
notes.md — notes export (human-readable Markdown)
attachments/ — attached files
manifest.json — metadata (doc title, compiler version, timestamp)

Credential security:
Your PAT is stored encrypted in this browser's IndexedDB. Key derivation: PBKDF2-SHA256 (210,000 iterations). Encryption: AES-256-GCM with a random 16-byte salt and 12-byte IV. The raw PAT never touches localStorage or sessionStorage.

Required PAT scopes:
GitHub / Forgejo: repo (contents read+write)
GitLab: api

🔒 Workspace Locked

Enter your workspace passphrase to decrypt the stored token.
